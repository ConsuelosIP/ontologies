DissolutionPeriodc back to ToC or Class ToC

IRI: http://parliament.uk/ontologies/time-period/DissolutionPeriod

The period between parliament periods during which Parliament is dissolved and does not exist. During this time there are no members, no committees etc. Dissolution periods are the same across both houses.
has super-classes
TimePeriodc

Eventc back to ToC or Class ToC

IRI: http://parliament.uk/ontologies/time-period/Event

A discrete event involving agents, locations and / or outputs. Events frequently initiate changes between time periods. Examples might be: State Opening, coronation, abdication, general election, prorogation, dissolution, recall.

ParliamentPeriodc back to ToC or Class ToC

IRI: http://parliament.uk/ontologies/time-period/ParliamentPeriod

The period of a single parliament between the state opening of the first session and dissolution. Defined as 5 years since the Fixed Term Parliament Act except in exceptional circumstances. Parliament periods are the same across both houses.

PorogationPeriodc back to ToC or Class ToC

IRI: http://parliament.uk/ontologies/time-period/PorogationPeriod

The period during which Parliament is inactive between prorogation announcement and State Opening. During this time there are still members but nearly all Parliamentary business stops. Prorogation periods are the same across both houses.

Sessionc back to ToC or Class ToC

IRI: http://parliament.uk/ontologies/time-period/Session

The period during which Parliament is active between any State Opening and prorogation or dissolution. A session typically lasts for one year though isn't bound by that. Any legislation introduced in a session must be passed in both houses during that session, unless it is carried over. Session periods are the same across both houses.
has super-classes
TimePeriodc
is in domain of
hasRecessop, hasTermop
is in range of
hasSessionop
