UK Parliament Ontologies

Table of Contents

  1. Agency ontology

    Provides a basic model of people, groups, memberships, positions and roles alongside contact details

  2. Concept ontology

    SKOS-like model for library subject headings (concepts)

  3. Core ontology

    Provides a set of common properties available for use by other models. Includes modelling for basic web links and alternative identifier schemes

  4. Election ontology

    Basic model for individual elections (e.g. a constituency election) and groups of elections (e.g. a general election) alongside some electorate and constituency modelling

  5. House membership ontology

    Provides a model to assign a member to a house (e.g. House of Commons or House of Lords) via a house seat alongside some links to interruptions of incumbency (leave of absence etc)

  6. Parliamentary groups ontology

    Adds specialisation to the agency model to cope with Parliament specific groups and membership properties

  7. Place ontology

    Model for geographic spaces
